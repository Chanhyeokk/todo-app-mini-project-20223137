# 나만의 오늘의 할 일 (Personal Todo App)

**나만의 계정을 만들고, 나만의 일정을 프라이빗하게 관리**할 수 있는 Todo 리스트 애플리케이션입니다. 

## 서비스 접속하기
- **Live Demo:** [https://todo-app-mini-project-20223137.vercel.app/] 

## 제공하는 기능

### 1. 개인 계정 (회원가입 & 로그인)
- **프라이빗 리스트:** 아이디와 비밀번호로 나만의 계정을 생성하세요. 내가 작성한 할 일은 나만 볼 수 있고, 나만 지울 수 있습니다.
- **안전한 보안:** 비밀번호는 강력하게 암호화되어 저장되며, 로그인 상태는 안전하게 유지됩니다.

### 2. 매일 반복되는 루틴 관리 (매일 할 일)
- 매일매일 해야 하는 양치, 운동, 영양제 챙겨 먹기 같은 루틴이 있을 경우 사용해 보세요.
- 추가 전, '매일 반복'을 체크하면, 일반 할 일과 섞이지 않도록 **노란색 전용 섹션**에 따로 모아서 보여줍니다.

### 3. 절대 잊으면 안 되는 일 (중요한 할 일)
- 오늘 안에 반드시 끝내야 하는 중요한 프로젝트나 약속이 있다면 '꼭 해야할 일'로 지정해 보세요.
- 글씨가 **진하게(Bold)** 강조되고, 수많은 할 일 목록 중에서도 항상 **가장 위쪽으로 자동으로 올라와서** 절대 놓치지 않게 도와줍니다.

### 4. 부드럽고 직관적인 디자인
- 차갑고 딱딱한 앱 대신, 부드러운 색감과 둥근 디자인을 적용했습니다.
- 직관적인 체크박스와 아이콘들로 일정을 관리하는 소소한 즐거움을 느껴보세요.

## 기술 스택
* **Frontend:** React, Vite, Tailwind CSS v3, Lucide Icons
* **Backend:** Node.js, Express.js, JSON Web Token(JWT), bcryptjs
* **Database:** MongoDB Atlas (Mongoose)
* **Deployment:** Vercel
